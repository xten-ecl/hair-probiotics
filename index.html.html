구글 시트 동기화 설정

구글 시트 Apps Script 웹앱 URL을 입력하면 PC와 모바일이 자동 동기화됩니다. 비워두면 이 기기에만 저장됩니다.
📋 설정 방법
1. 새 구글 시트 생성 → 확장 프로그램 → Apps Script
2. 안내받은 코드(.gs 파일) 붙여넣기 → 저장
3. 배포 → 새 배포 → 웹앱 선택
4. 액세스: 모든 사용자, 실행: 나
5. 배포 후 받은 URL을 위에 붙여넣기
✓ 저장됨
